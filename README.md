

# Onepager Website


Eine moderne **One-Page-Website**, erstellt mit **HTML** und **CSS**, die Text und Bilder in einem flüssigen und ansprechenden Design präsentiert.

---

## Beschreibung


Dieses Projekt ist eine Einseiten-Website (Onepager) mit mehreren Abschnitten:  

- **Hero-Sektion**: Einführung mit Titel und zentralem Text.  
- **Zweispalten-Sektionen**: Text und Bild nebeneinander, abwechselnd für ein ausgewogenes Layout.  
- **Hintergrundbild-Sektion**: Text über einem Bild mit Blur-Effekt (Unschärfe).  
- **Dreispalten-Sektion**: Informationsblöcke nebeneinander.  
- **Liste**: Elemente in stylischen Karten dargestellt.  
- **Großer Text**: zentrierter Textblock mit größerer Schrift.  
- **Kontakt-Sektion**: abschließender Bereich mit Kontaktinformationen.  

Das Design verwendet **Verläufe**, **weiche Schatten** und eine **Farbpalette von Blau-Violett bis Dunkel**, für ein modernes und elegantes Erscheinungsbild.
